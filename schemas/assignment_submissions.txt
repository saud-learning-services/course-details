assignment_id (integer, optional): The submission's assignment id,
assignment (Assignment, optional): The submission's assignment (see the assignments API) (optional),
course (Course, optional): The submission's course (see the course API) (optional),
attempt (integer, optional): This is the submission attempt number.,
body (string, optional): The content of the submission, if it was submitted directly in a text field.,
grade (string, optional): The grade for the submission, translated into the assignment grading scheme (so a letter grade, for example).,
grade_matches_current_submission (boolean, optional): A boolean flag which is false if the student has re-submitted since the submission was last graded.,
html_url (string, optional): URL to the submission. This will require the user to log in.,
preview_url (string, optional): URL to the submission preview. This will require the user to log in.,
score (number, optional): The raw score,
submission_comments (array[SubmissionComment], optional): Associated comments for a submission (optional),
submission_type (string, optional) = ['online_text_entry' or 'online_url' or 'online_upload' or 'media_recording' or 'student_annotation']: The types of submission ex: ('online_text_entry'|'online_url'|'online_upload'|'media_recording'|'student_annotation'),
submitted_at (datetime, optional): The timestamp when the assignment was submitted,
url (string, optional): The URL of the submission (for 'online_url' submissions).,
user_id (integer, optional): The id of the user who created the submission,
grader_id (integer, optional): The id of the user who graded the submission. This will be null for submissions that haven't been graded yet. It will be a positive number if a real user has graded the submission and a negative number if the submission was graded by a process (e.g. Quiz autograder and autograding LTI tools). Specifically autograded quizzes set grader_id to the negative of the quiz id. Submissions autograded by LTI tools set grader_id to the negative of the tool id.,
graded_at (datetime, optional),
user (User, optional): The submissions user (see user API) (optional),
late (boolean, optional): Whether the submission was made after the applicable due date,
assignment_visible (boolean, optional): Whether the assignment is visible to the user who submitted the assignment. Submissions where `assignment_visible` is false no longer count towards the student's grade and the assignment can no longer be accessed by the student. `assignment_visible` becomes false for submissions that do not have a grade and whose assignment is no longer assigned to the student's section.,
excused (boolean, optional): Whether the assignment is excused. Excused assignments have no impact on a user's grade.,
missing (boolean, optional): Whether the assignment is missing.,
late_policy_status (string, optional): The status of the submission in relation to the late policy. Can be late, missing, none, or null.,
points_deducted (number, optional): The amount of points automatically deducted from the score by the missing/late policy for a late or missing assignment.,
seconds_late (number, optional): The amount of time, in seconds, that an submission is late by.,
workflow_state (string, optional) = ['graded' or 'submitted' or 'unsubmitted' or 'pending_review']: The current state of the submission,
extra_attempts (number, optional): Extra submission attempts allowed for the given user and assignment.,
anonymous_id (string, optional): A unique short ID identifying this submission without reference to the owning user. Only included if the caller has administrator access for the current account.,
posted_at (datetime, optional): The date this submission was posted to the student, or nil if it has not been posted.,
read_status (string, optional) = ['read' or 'unread']: The read status of this submission for the given user (optional). Including read_status will mark submission(s) as read.
