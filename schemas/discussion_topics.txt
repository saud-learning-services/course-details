id (integer, optional): The ID of this topic.,
title (string, optional): The topic title.,
message (string, optional): The HTML content of the message body.,
html_url (string, optional): The URL to the discussion topic in canvas.,
posted_at (datetime, optional): The datetime the topic was posted. If it is null it hasn't been posted yet. (see delayed_post_at),
last_reply_at (datetime, optional): The datetime for when the last reply was in the topic.,
require_initial_post (boolean, optional): If true then a user may not respond to other replies until that user has made an initial reply. Defaults to false.,
user_can_see_posts (boolean, optional): Whether or not posts in this topic are visible to the user.,
discussion_subentry_count (integer, optional): The count of entries in the topic.,
read_state (string, optional) = ['read' or 'unread']: The read_state of the topic for the current user, 'read' or 'unread'.,
unread_count (integer, optional): The count of unread entries of this topic for the current user.,
subscribed (boolean, optional): Whether or not the current user is subscribed to this topic.,
subscription_hold (string, optional) = ['initial_post_required' or 'not_in_group_set' or 'not_in_group' or 'topic_is_announcement']: (Optional) Why the user cannot subscribe to this topic. Only one reason will be returned even if multiple apply. Can be one of: 'initial_post_required': The user must post a reply first; 'not_in_group_set': The user is not in the group set for this graded group discussion; 'not_in_group': The user is not in this topic's group; 'topic_is_announcement': This topic is an announcement,
assignment_id (integer, optional): The unique identifier of the assignment if the topic is for grading, otherwise null.,
delayed_post_at (datetime, optional): The datetime to publish the topic (if not right away).,
published (boolean, optional): Whether this discussion topic is published (true) or draft state (false),
lock_at (datetime, optional): The datetime to lock the topic (if ever).,
locked (boolean, optional): Whether or not the discussion is 'closed for comments'.,
pinned (boolean, optional): Whether or not the discussion has been 'pinned' by an instructor,
locked_for_user (boolean, optional): Whether or not this is locked for the user.,
lock_info (LockInfo, optional): (Optional) Information for the user about the lock. Present when locked_for_user is true.,
lock_explanation (string, optional): (Optional) An explanation of why this is locked for the user. Present when locked_for_user is true.,
user_name (string, optional): The username of the topic creator.,
topic_children (array[integer], optional): DEPRECATED An array of topic_ids for the group discussions the user is a part of.,
group_topic_children (array[object], optional): An array of group discussions the user is a part of. Fields include: id, group_id,
root_topic_id (integer, optional): If the topic is for grading and a group assignment this will point to the original topic in the course.,
podcast_url (string, optional): If the topic is a podcast topic this is the feed url for the current user.,
discussion_type (string, optional) = ['side_comment' or 'threaded']: The type of discussion. Values are 'side_comment', for discussions that only allow one level of nested comments, and 'threaded' for fully threaded discussions.,
group_category_id (integer, optional): The unique identifier of the group category if the topic is a group discussion, otherwise null.,
attachments (array[FileAttachment], optional): Array of file attachments.,
permissions (object, optional): The current user's permissions on this topic.,
allow_rating (boolean, optional): Whether or not users can rate entries in this topic.,
only_graders_can_rate (boolean, optional): Whether or not grade permissions are required to rate entries.,
sort_by_rating (boolean, optional): Whether or not entries should be sorted by rating.
