id (integer, optional): the unique identifier for the module item,
module_id (integer, optional): the id of the Module this item appears in,
position (integer, optional): the position of this item in the module (1-based),
title (string, optional): the title of this item,
indent (integer, optional): 0-based indent level; module items may be indented to show a hierarchy,
type (string, optional) = ['File' or 'Page' or 'Discussion' or 'Assignment' or 'Quiz' or 'SubHeader' or 'ExternalUrl' or 'ExternalTool']: the type of object referred to one of 'File', 'Page', 'Discussion', 'Assignment', 'Quiz', 'SubHeader', 'ExternalUrl', 'ExternalTool',
content_id (integer, optional): the id of the object referred to applies to 'File', 'Discussion', 'Assignment', 'Quiz', 'ExternalTool' types,
html_url (string, optional): link to the item in Canvas,
url (string, optional): (Optional) link to the Canvas API object, if applicable,
page_url (string, optional): (only for 'Page' type) unique locator for the linked wiki page,
external_url (string, optional): (only for 'ExternalUrl' and 'ExternalTool' types) external url that the item points to,
new_tab (boolean, optional): (only for 'ExternalTool' type) whether the external tool opens in a new tab,
completion_requirement (CompletionRequirement, optional): Completion requirement for this module item,
content_details (ContentDetails, optional): (Present only if requested through include[]=content_details) If applicable, returns additional details specific to the associated object,
published (boolean, optional): (Optional) Whether this module item is published. This field is present only if the caller has permission to view unpublished items.
