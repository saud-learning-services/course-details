id (integer, optional): the unique identifier for the module,
workflow_state (string, optional) = ['active' or 'deleted']: the state of the module: 'active', 'deleted',
position (integer, optional): the position of this module in the course (1-based),
name (string, optional): the name of this module,
unlock_at (datetime, optional): (Optional) the date this module will unlock,
require_sequential_progress (boolean, optional): Whether module items must be unlocked in order,
prerequisite_module_ids (array[integer], optional): IDs of Modules that must be completed before this one is unlocked,
items_count (integer, optional): The number of items in the module,
items_url (string, optional): The API URL to retrive this module's items,
items (array[ModuleItem], optional): The contents of this module, as an array of Module Items. (Present only if requested via include[]=items AND the module is not deemed too large by Canvas.),
state (string, optional) = ['locked' or 'unlocked' or 'started' or 'completed']: The state of this Module for the calling user one of 'locked', 'unlocked', 'started', 'completed' (Optional; present only if the caller is a student or if the optional parameter 'student_id' is included),
completed_at (datetime, optional): the date the calling user completed the module (Optional; present only if the caller is a student or if the optional parameter 'student_id' is included),
publish_final_grade (boolean, optional): if the student's final grade for the course should be published to the SIS upon completion of this module,
published (boolean, optional): (Optional) Whether this module is published. This field is present only if the caller has permission to view unpublished modules.
